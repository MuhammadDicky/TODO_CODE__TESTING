1. Hubungkan halaman home dengan new_todo. Ketika user klik tombol +, maka user akan diarahkan kehalaman
   input todo.
2. Dihalaman new_todo, ketika input new todo diisi dan user menekan tombol +, maka user akan diarahkan kembali
   kehalaman home dengan todo yang sebelum nya diinput tadi berada pada baris paling atas.
3. Fungsi kan checkbox pada daftar todo dimana jika user menekan checkbox maka todo tadi akan selesai dan berada pada
   barisan daftar todo yang sudah dicek sebelumnya.
4. Fungsikan tombol hapus pada baris data todo.
5. Fungsikan pencarian todo pada input searchbox pada halaman home.
6. Total check digantikan dengan jumlah dari todo yang aktif dan telah berstatus selesai. sedangkan total digantikan
   dengan total todo yang saat ini telah diinput.

requirement: gunakan getx dependency sebagai state management.

estimasi: 1 jam 5 menit.
